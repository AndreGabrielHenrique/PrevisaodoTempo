## Resumo do Projeto

Aplicação frontend em React que fornece previsões meteorológicas para localidades no Brasil. Ela utiliza a API do OpenWeatherMap para obter clima atual e previsão para os próximos 5 dias, e a API do IBGE para validação e listagem de municípios. A interface suporta geolocalização do usuário, busca por nome de cidade com autocompletar e apresenta dados como temperatura, sensação térmica, umidade e pressão.

Para rodar em desenvolvimento:

```bash
npm install
npm run dev
```
